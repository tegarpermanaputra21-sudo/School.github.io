# University Info Web / Website Informasi Universitas

(EN)
### Description
This website is a portfolio project built from scratch while following a tutorial by GreatStack Dev. All code is written independently, with custom animations and styling added. Some images used in the project are taken from the playlist description of GreatStack Dev. The project is intended for learning web development and showcasing a portfolio.

### Key Features
 - Home page with general university description
 - Faculties and programs page
 - Facilities and contact page
 - Custom animations and styling

### Reference / Source
 - Inspired by GreatStack Dev (https://www.youtube.com/watch?v=oYRda7UtuhA&list=PLjwm_8O3suyP5kGKmwS_DM0Hs1j7fshi5)
 - Images or assets taken from the playlist description of GreatStack Dev

### Note
Project created for portfolio and learning purposes. All university content is neutral and does not use official university materials.
